Private/Public event booking calendar for Newport Roxy.

== Description ==

Roxy Event Booking supports:

* Private and public event booking
* WooCommerce checkout integration
* Business invoice bookings
* Pizza ordering
* Bulk concession ordering
* Pricing configuration
* Availability and scheduling logic
* My Account booking views
* Admin booking management
* Sling staffing integration
* GitHub auto-updates

== Features ==

Booking Features

* Date/time booking
* Guest count
* Extra hours
* Movie or live event
* Private or public event

Payment Options

* Personal pay now
* Business pay now
* Business invoice

Pizza Ordering

* Quantity selection
* Order notes
* Configurable pricing
* Reminder system

Bulk Concessions

* Bulk popcorn
* Bulk soda
* 25 minimum
* Configurable pricing

Sling Integration

* Automatic shift creation
* Pizza notes
* Bulk concession notes

== Installation ==

1. Upload plugin ZIP
2. Activate plugin
3. Configure settings
4. Create GitHub release for updates
