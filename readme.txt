Private/Public event booking calendar for Newport Roxy.

== Description ==

Roxy Event Booking supports:

* Private and public event booking
* WooCommerce checkout integration
* Availability and scheduling logic
* My Account booking views
* Admin booking management
* Sling staffing integration
* GitHub auto-updates

== Installation ==

1. Upload the plugin ZIP to WordPress.
2. Activate the plugin.
3. Configure settings and booking product.
4. Create a GitHub release tag beginning with v for packaged updates.
